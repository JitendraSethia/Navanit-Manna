🔧 OPTIONAL: GOOGLE REVIEWS AUTO-FETCH

Want reviews to update automatically from Google Maps?

Option A - PHP (for shared hosting):
  1. Get Google API key
  2. Edit api-reviews.php
  3. Upload to server

Option B - Node.js (for VPS):
  1. Run: npm install
  2. Copy .env.example to .env
  3. Add API key to .env
  4. Run: npm start

See SETUP-GUIDE.md for detailed instructions.
