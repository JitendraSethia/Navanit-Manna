## Image Guidelines

- **Format**: JPG or PNG
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**: 
  - Doctor photo: 800x800px (square)
  - Transformations: 1200x900px (landscape)
  - Video thumbnails: 1280x720px (16:9 ratio)

## Tips

- Use high-quality images
- Ensure good lighting
- Compress images before uploading
- Maintain patient privacy (get consent for before/after photos)
- Use consistent styling across images
