# 🚀 Hosting Guide - Get Your Website Live in 5 Minutes!

## 🎯 Best Options (All FREE!)

### Option 1: Netlify (RECOMMENDED - Easiest!)

**Why Netlify?**
- 100% Free
- Drag & drop deployment
- Automatic HTTPS
- Custom domain support
- Takes 2 minutes!

**Steps:**

1. **Go to Netlify**
   - Visit: https://www.netlify.com/
   - Click "Sign up" (use GitHub, Google, or Email)

2. **Deploy Your Site**
   - Click "Add new site" → "Deploy manually"
   - Drag the ENTIRE folder (N Manna) into the upload area
   - Wait 30 seconds... Done! ✅

3. **Get Your Link**
   - You'll get a URL like: `https://random-name-123.netlify.app`
   - Click "Site settings" → "Change site name" to customize
   - Example: `dr-n-manna-clinic.netlify.app`

4. **Add Custom Domain (Optional)**
   - Buy a domain (like `drmannaclinic.com`)
   - In Netlify: "Domain settings" → "Add custom domain"
   - Follow instructions

**That's it! Your site is live! 🎉**

4. Import your folder or drag & drop

6. Get your link: `https://dr-manna-clinic.vercel.app`

---

### Option 3: GitHub Pages (Free Forever!)

**Steps:**

1. **Create GitHub Account**
   - Go to: https://github.com/
   - Sign up (free)

2. **Create Repository**
   - Click "+" → "New repository"
   - Name it: `dr-manna-clinic`
   - Make it Public
   - Click "Create repository"

3. **Upload Files**
   - Click "uploading an existing file"
   - Drag ALL your files (index.html, styles.css, etc.)
   - Include the `images` folder
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to "Settings" → "Pages"
   - Source: "Deploy from a branch"
   - Branch: "main" → "/ (root)"
   - Click "Save"

5. **Get Your Link**
   - Wait 2 minutes
   - Your site: `https://yourusername.github.io/dr-manna-clinic/`

---

### Option 4: InfinityFree (Traditional Hosting)

**If you want cPanel/FTP hosting:**

2. Sign up (free)
3. Create account
4. Use File Manager or FTP
5. Upload all files to `htdocs` folder
6. Get subdomain: `yoursite.infinityfreeapp.com`

## 🆘 Troubleshooting

### Images Not Showing?
- Make sure `images` folder is uploaded
- Check file names match exactly
- Images must be in `images/` folder

### Site Not Loading?
- Wait 2-5 minutes after deployment
- Clear browser cache (Ctrl+F5)
- Try incognito mode

### Custom Domain Not Working?
- Wait 24-48 hours for DNS
- Check nameserver settings
- Verify domain is active
